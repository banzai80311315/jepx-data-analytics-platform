# JEPXデータ分析基盤  
## 開発スケジュール

本プロジェクトは **約5か月** で開発を進める計画とする。  

目的は、日本卸電力取引所（JEPX）の電力価格データを収集・蓄積・分析し、  
電力価格スパイクの統計的特徴を分析する **データ分析基盤** を構築することである。

本プロジェクトでは **データ分析を中心とした開発方針** を採用する。

```
環境構築
↓
データ取得
↓
データ加工
↓
データ分析
↓
アプリ自動実行
```

システム開発は目的ではなく、**データ分析を支える基盤**として位置付ける。

