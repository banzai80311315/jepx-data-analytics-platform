# JEPXデータ分析基盤  
## 開発スケジュール

本プロジェクトは **5か月** で開発を進める。  

目的は、日本卸電力取引所（JEPX）の30分電力価格データを分析し、  
ブラウザ上から期間・エリア・分析手法を指定して実行できる  
**インタラクティブ分析アプリケーション** を構築することである。  

本アプリは、電力価格スパイクの研究を進めるための  
**データ分析基盤** として設計する。

---

# 開発方針

本プロジェクトは、以下の順序で進める。

```text
環境構築
↓
データ読み込み基盤
↓
MVP分析アプリ
↓
分析機能拡張
↓
運用・自動化
```

システム開発は目的ではなく、  
**電力価格データを継続的に分析するための研究基盤を構築すること** を主目的とする。

## Phase2 データ読み込み・前処理機能の実装

### 目的
JEPX年度CSVを読み込み、分析可能なDataFrameへ整形する。

### 作業内容

- JEPX CSV の配置ルールを決定
- `loader.py` の実装
  - CSV読み込み
  - DataFrame化
- `processor.py` の実装
  - 日時列の変換
  - エリア列の整理
  - 必要列の抽出
  - 分析期間による絞り込み
- 生データと加工データの保存ルール決定

### 処理フロー

```text
JEPX CSV
↓
loader.py
↓
processor.py
↓
分析用DataFrame
```

### 成果物

- `loader.py`
- `processor.py`
- 加工済みデータの基本形
- 指定期間・指定エリアで抽出できる処理

---

## Phase3 MVP分析アプリの構築

### 目的
最低限の分析機能を持つ Streamlit アプリを完成させる。

### MVP機能

- 年度CSV読み込み
- 分析期間指定
- エリア選択
- 時系列可視化
- 30分単位の平均価格表示

### 作業内容

- `app.py` に画面を実装
- 入力UIの作成
  - start_date
  - end_date
  - area
- `analyzer.py` の実装
  - 時系列抽出
  - 時間帯平均
- `visualizer.py` の実装
  - 折れ線グラフ
  - 平均価格グラフ

### アプリ構造

```text
Browser
↓
Streamlit UI
↓
Analysis Controller
↓
Analysis Modules
↓
Processed Data
```

### 成果物

- ブラウザから操作できる MVP アプリ
- 基本的な時系列分析と可視化
- 「まず使えるもの」が完成した状態

## Phase4 分析機能の拡張

### 目的
研究に近づくための分析機能を追加する。

### 拡張機能

- ヒストグラム
- 対数分布表示
- スパイク検出
- エリア比較
- 分析結果の図保存

### 作業内容

- `analyzer.py` に分布分析処理を追加
- スパイク判定ロジック実装
  - 例: `price > mean + 3σ`
- `visualizer.py` に追加グラフを実装
- `outputs/figures/` に図を保存する処理追加

### 成果物

- 分布分析機能
- スパイク検出機能
- 研究用途に近い分析画面
- 図の保存機能
